# Cairn Roadmap

## v0.3 — Cross-Platform Foundation ✅
*Current state*

- [x] Electron desktop app (macOS, Windows, Linux)
- [x] PWA with offline support (Android, iOS)
- [x] Media upload and management
- [x] Audio recording in-browser
- [x] Relationship management in web UI
- [x] Rich markdown editor with live preview
- [x] Family tree visualization (SVG)
- [x] Timeline grouped by decade
- [x] Statistics dashboard
- [x] Dark mode (auto + manual)
- [x] GitHub Pages project site
- [x] Full frontend redesign (warm archive aesthetic)

## v0.4 — The Bridge

### Search & Intelligence
- [ ] Full-text search across all content (SQLite FTS5)
- [ ] Search result snippets with highlighting
- [ ] Semantic search for stories
- [ ] Autocomplete for names, tags, places

### Legacy Features
- [ ] Inactivity timer / dead man's switch
- [ ] Emergency access codes for successors
- [ ] Timed delivery of stories to loved ones
- [ ] "Time capsule" — scheduled release of memories

### Media Improvements
- [ ] Drag-and-drop media upload
- [ ] Image gallery view with lightbox
- [ ] Video playback in-app
- [ ] Bulk media import
- [ ] Face detection for auto-tagging people in photos

### Collaboration
- [ ] Import from JSON
- [ ] Merge two vaults (deduplication)
- [ ] Git-like sync between devices (via file or local network)
- [ ] Encrypted archive sharing

## v0.5 — The Network

### P2P & Distribution
- [ ] Peer-to-peer sync between family members
- [ ] Community archives (shared, governed vaults)
- [ ] Encrypted end-to-end sync
- [ ] Conflict resolution for concurrent edits

### AI & Prompts
- [ ] Story prompt engine ("Tell me about your childhood home")
- [ ] Audio transcription (local Whisper model)
- [ ] Automatic relationship suggestions
- [ ] Photo OCR for document scanning
- [ ] "On this day" memory reminders

### Platform Expansion
- [ ] Mobile app (React Native or Capacitor) for camera/audio access
- [ ] Auto-backup to user-controlled storage (Nextcloud, Syncthing)
- [ ] Native file picker integration in Electron
- [ ] System tray integration

## v1.0 — The Standard

### Core
- [ ] Plugin system for custom data types
- [ ] The Cairn Format Specification v1.0
- [ ] Comprehensive test suite (100% coverage)
- [ ] Security audit and hardening
- [ ] Performance optimization for large archives (100k+ stories)

### Ecosystems
- [ ] Reference implementation (this project)
- [ ] Web API for third-party tools
- [ ] Browser extension for clipping content into Cairn
- [ ] Obsidian plugin for bidirectional linking
- [ ] CLI-based CI/CD for automated archiving

### Community
- [ ] Localization (i18n) — top 10 languages
- [ ] Accessibility audit (WCAG 2.1 AA)
- [ ] Community template packs for specific use cases
  - Family history template
  - Oral history project template
  - Community culture preservation template
  - Personal journal template
- [ ] Starter kits for non-technical users

## The Long Horizon

- **Cairn Protocol** — open standard for personal archiving
- **Cairn Network** — opt-in distributed web of family archives
- **Cairn Foundation** — non-profit ensuring the project outlives any individual contributor
- **Cairn AI** — optional local AI companion for memory preservation
- **Cairn in education** — tools for oral history projects in schools
- **Cairn for endangered cultures** — specialized tools for language and tradition preservation

## How to contribute

See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to help with any of these goals.

The roadmap is a living document. If you want to champion a feature, open an issue, start a discussion, or submit a PR. Every contribution — code, design, docs, translation, testing, ideas — moves the cairn forward.
